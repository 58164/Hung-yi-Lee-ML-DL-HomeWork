## HW2

在baseline的基础上：

-   **对原始数据归一化**
-   dataloader加了个**WeightedRandomSampler**加权抽样（避免高频数据主导梯度）
-   **训练数据加了随机遮盖和随机dropout**
-   **训练数据加了一些噪声**

**这样baseline的score从0.49提升到了0.52**

然后考虑把**MLP**换成**RNN**

然后看了下公开的strong baseline，model非常简单，bilstm 的输出接到MLP就完了，但是为了更好的捕捉时序特征，需要把concat_nframes 和 rnn的 hidden_dim 开的很大，我为了短时间内做完这个作业，随便调了小参数就跑了。

近2h跑完，拿到了**0.75**的分数

进一步地调大参数应该可以拿到更高的分数，不过也没有必要了。
